# LegalForge

Court document intelligence for Claude Code. The first AI plugin for legal professionals.

## Features

### Skills
- **court-formatting** -- PD 32, CPR Part 22, Chancery Guide formatting rules
- **citation-validator** -- Neutral citations, statute refs, CPR rules, exhibit cross-refs
- **exhibit-manager** -- Numbering, cover pages, cross-references, renumbering
- **bundle-rules** -- E-bundle format, pagination, indexing, Chancery Guide compliance

### Agents
- **legal-reviewer** -- Comprehensive document review (structure, citations, exhibits, formatting)

## Install

```bash
claude plugin install legal-forge
```

## Why This Exists

300+ AI agents in the major directories. Zero for legal professionals. Built by a Litigant in Person from 152 BDD feature files encoding court rules as executable specifications.

## Jurisdictions

England and Wales courts (High Court, County Court, SCCO). Contributions for other jurisdictions welcome.

## Not Legal Advice

This plugin provides document formatting and quality assurance, not legal advice.
