# Court Bundle Assembly Rules

## E-Bundle Format (Chancery Guide Ch 15)
- Single PDF, searchable text (OCR scanned docs)
- Bookmarks for every document
- Hyperlinked index with page references
- A4 portrait, single continuous page numbering

## Standard Bundle Contents (in order)
1. Index (hyperlinked)
2. Claim Form / Application Notice
3. Statements of Case
4. Court Orders (chronological)
5. Witness Statements with exhibits
6. Expert Reports
7. Skeleton Arguments
8. Draft Order
9. Authorities Bundle (separate PDF)

## Pagination
- Single continuous sequence, every page numbered
- Page numbers must match PDF viewer page
- Index may use roman numerals

## Size
- E-bundles have no fixed page limit
- Core reading bundle: 20-30 pages
- Skeleton arguments: max 25 pages
- Authorities: max 10 unless justified

## Validation
- [ ] Single PDF, searchable, bookmarked
- [ ] Hyperlinked index
- [ ] Continuous page numbering
- [ ] All index documents present
- [ ] All exhibits have cover pages
- [ ] Correct document order
- [ ] Reasonable file size
