# Court Document Formatting Rules (England and Wales)

## Witness Statement Rules (PD 32)

### Structure
- Headed with case name, claim number, and court
- Numbered paragraphs throughout
- State maker's full name, address, and occupation
- State whether evidence is from own knowledge or belief (and source)
- Exhibits referenced as initials + number (e.g., SP1, SP2)

### Statement of Truth
Must include: "I believe that the facts stated in this witness statement are true. I understand that proceedings for contempt of court may be brought against anyone who makes, or causes to be made, a false statement in a document verified by a statement of truth without an honest belief in its truth."

### Formatting
- A4 portrait, minimum 3.5cm left margin, minimum 12pt font
- 1.5 or double line spacing, pages numbered consecutively
- Paragraphs numbered consecutively

## Exhibit Rules (PD 32.11-32.15)

### Cover Page Required
Every exhibit MUST have a front page stating:
- Exhibit number/mark (e.g., "This is exhibit SP1")
- Brief description
- "Referred to in the witness statement of [name]"
- Date of witness statement
- Case name

### Numbering
- Sequential from first to last
- Continue from previous statement if supplementary
- Bundle order matches witness statement reference order

## Skeleton Argument Rules (Chancery Guide Ch 16)
- Summary of case (1-2 paragraphs)
- Chronology (if not agreed, attach separately)
- Propositions of law with citations
- Submissions on facts
- Relief sought
- Maximum 25 pages (unless court permission)

## Bundle Rules (Chancery Guide Ch 15)
- PDF with bookmarks and hyperlinked index
- Searchable text (OCR if necessary)
- Single continuous page numbering
- A4 portrait

## Validation Checklist
- [ ] Case name and claim number in header
- [ ] Paragraphs numbered consecutively
- [ ] Statement of truth present and correctly worded
- [ ] All exhibits have cover pages
- [ ] Exhibit numbering sequential
- [ ] Font >= 12pt, left margin >= 3.5cm, line spacing >= 1.5
- [ ] Pages numbered, date present, signature block present
